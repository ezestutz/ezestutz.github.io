Sitio web de Capitán Sarmiento

Este proyecto se dió dentro del marco de una pasantía realizada en la Municipalidad de Capitán Sarmiento. El sitio fue desarrollado en HTML, CSS, con uso de Bootstrap, JavaScript y PHP, con el objetivo de darle al vecino la posibilidad de acceder a diversos servicios e información desde un único lugar. Cave destacar que la ciudad no tenía un sitio web, por lo tanto el trabajo también involucró crear y administrar una instancia EC2 en Amazon. Y crear el servidor Apache en la misma desde 0.

"Sugerencias y Reclamos"

También uno de los proyectos de la pasantía. Consiste en un sistema basado en osTicket, con profundas modificaciones estructurales y estéticas, en PHP (con consultas en SQL), HTML, CSS y JavaScript, para adaptarse al diseño del sitio web y las necesidades especificas del municipio. Con el objetivo de establecer una conexión más estrecha con los vecinos y de adaptarnos a la situación de cuarentena y aislamiento, propusimos este proyecto.

El sistema consta de un entrono para el vecino, en el que puede abrir reclamos o sugerencias y ver el estado de los que ya abrió; y un entorno para los responsables de contestar los reclamos y sugerencias. Una vez se abre un reclamo o sugerencia, le llegará un mail al responsable de responder, con un link para iniciar en el sistema y contestarlo.

"Atención al Vecino"

Una vez desarrollados las aplicaciones mencionadas anteriormente, desarrollamos una app para dispositivos Android, en Java, con el fin de hacer más práctico y simple el uso de los sistemas desarrollados. Esta app proporciona acceso al sistema "Sugerencias y Reclamos", además de una pantalla que ofrece enviar un mensaje por WhatsApp o realizar una llamada a un número de teléfono, para realizar algun turno en especifico y un acceso a un entorno, para realizar pagos.

"Seguridad Ciudadana"

También formé parte del desarrolo de otra aplicación en Java, para Android, capaz de enviar diferentes tipos de alertas de emergencias, tomando la ubicación actual del dispositivo y la información del usuario.
Dichas alertas son visualizadas en un centro de monitoreo desarrollado en ReactJS con un backend en NodeJS y MySQL, en donde suena una alarma y se marca en un mapa la ubicación de la emergencia, disparando a continuación un protocolo articulado por los servicios de emergencia municipales. Por cuestiones de seguridad no estoy autorizado a mostrar el entorno o proveer acceso a él.
